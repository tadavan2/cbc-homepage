## Typography

### Fonts
- **Headings/Logo**: Futura, "Futura PT", "Century Gothic", "AppleGothic", sans-serif
- **Body**: Space Grotesk, Montserrat, system fonts

## Navigation

### Desktop
- Glassmorphism blur effect
- Hidden until user scrolls
- Company name: CALIFORNIA (yellow) BERRY (red) CULTIVARS (dark green)

### Mobile
- Hamburger menu
- Slide-in panel from right
- 30px top padding before first menu item

---

## Animation Timing

### Intro Overlay
1. Hold state (wait for scroll)
2. On scroll: panels slide in (1s)
3. Fadeout overlay (1.5s)
4. Total intro duration: ~2.5s after scroll

### Hero Content
- Fade in triggered 0.5s after scroll
- 1.5s fade duration
